# 🌍 Quick Translator Chrome Extension

A simple Chrome Extension that allows you to **highlight text on any webpage and instantly translate it** into Vietnamese or Japanese.

---

## ✨ Features

- 🔥 Highlight text on any website to translate instantly
- 🌐 Supports English → Vietnamese (EN → VI)
- 🌐 Supports English → Japanese (EN → JP) (configurable)
- ⚡ Lightweight, no login required
- 🪟 Floating popup showing original text + translation
- 🚀 Works on all websites
